# 🍯 Donna Doceria | Alta Confeitaria

Este é o projeto de presença digital da **Donna Doceria**, uma doceria sofisticada baseada em Brasília. O site foi desenvolvido com foco na experiência do usuário, estética de luxo e performance otimizada através de uma abordagem **Mobile First**.

---

## ✨ Diferenciais do Projeto

* **Design de Luxo:** Paleta de cores baseada em tons de dourado (`#c5a059`), rosa profundo (`#e91e63`) e muito respiro visual.
* **Abordagem Mobile First:** Estrutura pensada primeiro para dispositivos móveis e escalada para desktop via Media Queries.
* **Interatividade Suave:**
    * **Logo Dinâmica:** Transição suave entre as versões preta e dourada no header.
    * **Scroll Progress Bar:** Barra de leitura sutil em `var(--deep-pink)` que acompanha o usuário no topo da página.
    * **Reveal Animations:** Elementos que surgem em cascata (*fade-in up*) ao carregar a página utilizando CSS Keyframes.
* **Performance:** Desenvolvido puramente com HTML5, CSS3 e Vanilla JavaScript, garantindo carregamento instantâneo e leveza.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica para melhor SEO e acessibilidade.
* **CSS3:** Uso de variáveis nativas (`:root`), Flexbox, CSS Grid e animações complexas.
* **Vanilla JS:** Lógica de scroll e manipulação do DOM sem dependência de bibliotecas externas.
* **Google Fonts:** Mix das fontes *Playfair Display* (Serifada para elegância) e *Montserrat* (Sans-serif para legibilidade).
